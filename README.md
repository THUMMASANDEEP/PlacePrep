# PlacePrep - Placement Tracker Dashboard

## Overview
PlacePrep is a Placement Tracker Dashboard designed to help students organize and monitor their placement preparation journey.

## Features
- Dashboard Overview
- Study Timer
- Habit Tracker
- Goal Management
- Company Tracker
- Interview Q&A
- Mock Tests
- Analytics & Progress Tracking
- Resources Section

## Technologies Used
- HTML
- CSS
- JavaScript
